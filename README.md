# Urlish [![Build Status](https://travis-ci.org/rgratwick/urlish.png)](https://travis-ci.org/rgratwick/urlish)

Gem based on Railscast #301-303

## Installation

Add this line to your application's Gemfile:

    gem 'urlish'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install urlish
